## 詳細な説明文

capsicum は、Mastodon と Misskey に対応した Fediverse クライアントアプリです。

お好みのサーバーにログインして、タイムラインの閲覧や投稿をお楽しみいただけます。複数のサーバー・アカウントを登録して切り替えることも可能です。

主な機能:
- タイムライン閲覧（ホーム / ローカル / 連合 / リスト / ハッシュタグ）と無限スクロール
- テキスト・メディア・投票付き投稿（公開範囲選択、CW 対応）
- リプライ・ブースト・お気に入り・ブックマーク
- Misskey リアクション（絵文字ピッカー付き）
- リアルタイム更新（WebSocket ストリーミング）
- MFM / HTML / カスタム絵文字の描画
- 通知一覧・検索・プロフィール表示
- フォロー / ミュート / ブロック操作
- 引用投稿・プレビューカードの表示
- メディア添付（画像・動画）とメディアビューア
- ワードフィルタの反映
- mulukhiya-toot-proxy（モロヘイヤ）連携

capsicum はオープンソース（AGPL-3.0）です。
https://capsicum.shrieker.net

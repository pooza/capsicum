# ストア掲載情報

## アプリ名

capsicum

## 短い説明文（80文字以内）

Mastodon / Misskey 対応の Fediverse クライアント。複数アカウントに対応。

## 詳細な説明文

capsicum は、Mastodon と Misskey に対応した Fediverse クライアントアプリです。

お好みのサーバーにログインして、タイムラインの閲覧や投稿をお楽しみいただけます。複数のサーバー・アカウントを登録して切り替えることも可能です。

主な機能:
- タイムライン閲覧（ホーム / ローカル / 連合）と無限スクロール
- テキスト・メディア付き投稿（公開範囲選択、CW 対応）
- 通知一覧
- 投稿の検索（アカウント / ハッシュタグ / 全文検索）
- ユーザープロフィールの表示
- お気に入り / ブースト / ブックマーク
- Misskey リアクション
- リアルタイム更新（ストリーミング対応）
- カスタム絵文字の表示
- ワードフィルタの反映

capsicum はオープンソース（AGPL-3.0）です。
https://github.com/pooza/capsicum
